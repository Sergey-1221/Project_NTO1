Перед открытием нашего приложения,
рекомендуется закрыть приложения других участников олимпиады.
Запуск:
Логин: admin 
Пароль: admin 
Открыть приложение [app_name1]
Если сайт не открылся автоматически,
Перейти по адресу http://127.0.0.1:8000

Также имеется резервное приложение [app_name1] 
(если первое по какой-либо причине не работает)
Адрес резервного приложения http://127.0.0.1:8080

Так как серевр запускается локально,
 то для его работы нужно не закрывать открывшуюся консоль (скрин окна)

В продакшн решении все будет на облачных серверах.
И пользователю для доступа к системе нужно будет ввести только домен сайта.
